1. TransformMyDelta

1.1 配置TransformMyDelta

services.AddOpenApi(options => options.TransformMyDelta());

1.2 转化前

"MyDeltaOfTodoItem": { }

1.3 使用TransformMyDelta后

  "type": "object",
  "properties": {
    "id": {
      "type": "integer",
      "format": "int64"
    },
    "name": {
      "type": "string",
      "nullable": true
    },
    "isComplete": {
      "type": "boolean"
    },
    "remark": {
      "type": "string",
      "nullable": true
    }
  }
}
